# 🛒 Customer Shopping Trends Analysis

## 📌 Project Overview

This project analyzes customer shopping behavior using SQL, Python, and Power BI. The objective is to identify purchasing trends, customer segments, category performance, discount effectiveness, and sales patterns to support business decision-making.

---

## 🎯 Business Objectives

- Analyze customer purchasing behavior.
- Identify high-value customer segments.
- Evaluate product category performance.
- Measure discount and promotion effectiveness.
- Discover sales trends.

---

## 🛠️ Technologies Used

- Python
- SQL (MySQL)
- Power BI
- Pandas
- NumPy
- Excel

---

## 📊 Analysis Performed

- Customer Segmentation
- Category-wise Sales Analysis
- Age Group Analysis
- Gender Analysis
- Purchase Distribution
- Discount Analysis
- Revenue Analysis

---

## 📈 Business Insights

- Identified top-selling product categories.
- Evaluated customer spending patterns.
- Measured promotional campaign performance.
- Compared customer purchasing behavior across age groups.
